# ⚛️ React Guide: Interactive Journey

Um guia interativo e visual sobre o ecossistema React, explorando desde sua trajetória histórica até conceitos fundamentais como JSX, Virtual DOM e Hooks.

🔗 **Acesse o projeto:** [https://react-guide-roan.vercel.app/](https://react-guide-roan.vercel.app/)

## 🚀 Tecnologias
- **React 18** (Functional Components & Hooks)
- **CSS3 Moderno** (Variáveis, Grid, Flexbox e Animações)
- **Canvas API** (Fundo de partículas interativo)
- **Intersection Observer API** (Animações de scroll)

## 💡 Funcionalidades
- **Linha do Tempo:** Evolução do React de 2011 até a era moderna.
- **Explorador de Conceitos:** Cards interativos que exibem snippets de código explicativos.
- **Demo de Hooks:** Exemplo prático em tempo real do hook `useState`.
- **Mapeamento de Ecossistema:** Visão geral das principais ferramentas (Next.js, Tailwind, Zustand, etc).
- **Interface Responsiva:** Design "Aurora" otimizado para dispositivos móveis e desktops.

## 🛠️ Como rodar localmente
1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Inicie o servidor de desenvolvimento: `npm run dev`.
